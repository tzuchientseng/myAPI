API Documentation

1. 計算平均數信賴區間 (Confidence Interval) - /CI

描述: 計算一組數據的平均數信賴區間

HTTP 方法: POST

URL: /CI

Request Parameters:
Name Type Description Required Default
data array 欲計算的數據列表 Yes None
CI float 信賴水準 (Confidence Level) No 95.0
Request Example:
curl -X POST -H "Content-Type: application/json" -d "{\"data\": [10, 20, 30, 40, 50], \"CI\": 90}" https://my-api-project-83a1e4f4e00f.herokuapp.com/CI
Response Example:
{
  "confidence_level": 90,
  "mean": 30.0,
  "lower_bound": 17.6,
  "upper_bound": 42.4
}
2. 計算二項信賴區間 (Proportion Confidence Interval) - /PCI

描述: 計算一組二項數據的信賴區間

HTTP 方法: POST

URL: /PCI

Request Parameters:
Name Type Description Required Default
successes integer 成功次數 Yes None
trials integer 試驗次數 Yes None
CI float 信賴水準 (Confidence Level) No 95.0
Request Example:
curl -X POST -H "Content-Type: application/json" -d "{\"successes\": 40, \"trials\": 100, \"CI\": 95}" https://my-api-project-83a1e4f4e00f.herokuapp.com/PCI
Response Example:
{
  "confidence_level": 95,
  "lower_bound": 0.3024,
  "upper_bound": 0.4976
}

4. 雙樣本平均數 t 檢定 (Two-Sample T-Test) - /TwoSampleTTest

描述: 比較兩組數據的平均數是否存在顯著差異

HTTP 方法: POST

URL: /TwoSampleTTest

Request Parameters:
Name Type Description Required Default
data1 array 第一組數據列表 Yes None
data2 array 第二組數據列表 Yes None
alpha float 顯著水準 (Significance Level) No 0.05
Request Example:
curl -X POST -H "Content-Type: application/json" -d "{\"data1\": [10, 20, 30, 40, 50], \"data2\": [5, 15, 25, 35, 45], \"alpha\": 0.05}" https://my-api-project-83a1e4f4e00f.herokuapp.com/TwoSampleTTest
Response Example:
{
  "null_hypothesis": "μ1 = μ2",
  "alternative_hypothesis": "μ1 ≠ μ2",
  "t_statistic": 0.743,
  "p_value": 0.485,
  "result": "Fail to reject the null hypothesis at α = 0.05 level."
}
