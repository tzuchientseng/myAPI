API Documentation

1. 計算平均數信賴區間 (Confidence Interval) - /CI

描述: 計算一組數據的平均數信賴區間

HTTP 方法: POST

URL: /CI

Request Parameters:
Name Type Description Required Default
data array 欲計算的數據列表 Yes None
CI float 信賴水準 (Confidence Level) No 95.0
Request Example:
curl -X POST -H "Content-Type: application/json" -d "{\"data\": [10, 20, 30, 40, 50], \"CI\": 90}" https://my-api-project-83a1e4f4e00f.herokuapp.com/CI
Response Example:
{
  "confidence_level": 90,
  "mean": 30.0,
  "lower_bound": 17.6,
  "upper_bound": 42.4
}

4. 雙樣本平均數 t 檢定 (Two-Sample T-Test) - /TwoSampleTTest

描述: 比較兩組數據的平均數是否存在顯著差異

HTTP 方法: POST

URL: /TwoSampleTTest

Request Parameters:
Name Type Description Required Default
data1 array 第一組數據列表 Yes None
data2 array 第二組數據列表 Yes None
alpha float 顯著水準 (Significance Level) No 0.05
Request Example:
curl -X POST -H "Content-Type: application/json" -d "{\"data1\": [10, 20, 30, 40, 50], \"data2\": [5, 15, 25, 35, 45], \"alpha\": 0.05}" https://my-api-project-83a1e4f4e00f.herokuapp.com/TwoSampleTTest
Response Example:
{
  "result": {
    "null_hypothesis": "μ1 = μ2",
    "alternative_hypothesis": "μ1 ≠ μ2",
    "t_statistic": 0.743,
    "p_value": 0.485,
    "decision": "Fail to reject the null hypothesis at α = 0.05 level."
  }
}

1. 線性回歸 (Linear Regression)

描述: 線性回歸假設銷售量和單筆營利之間存在線性關係。

適用場景: 當數據關係簡單且接近線性時，使用線性回歸效果較好。

特性:
優點 缺點
模型簡單，易於解釋，適合處理線性數據。 無法捕捉數據中的非線性關係。
計算效率高。 對複雜模式不適應。
Request Example:
curl -X POST -H "Content-Type: application/json" -d "{\"data\": [10, 20, 30, 40, 50]}" https://my-api-project-83a1e4f4e00f.herokuapp.com/linear_regression
Response Example:
{
    "max_profit": 11.46,
    "max_revenue": 1022.34
}
2. 多項式回歸 (Polynomial Regression)

描述: 多項式回歸擴展了線性回歸的能力，可以捕捉非線性數據關係。

適用場景: 當數據顯示曲線關係時，多項式回歸可以捕捉非線性特徵。

特性:
優點 缺點
適合捕捉非線性模式，能處理更複雜的曲線數據。 容易過擬合，模型變得複雜時泛化能力下降。
Request Example:
curl -X POST -H "Content-Type: application/json" -d "{\"data\": [10, 20, 30, 40, 50]}" https://my-api-project-83a1e4f4e00f.herokuapp.com/polynomial_regression
{
    "max_profit": 12.68,
    "max_revenue": 1139.11
}
3. 決策樹回歸 (Decision Tree Regression)

描述: 決策樹回歸通過將數據分割成區間，並在每個區間內進行預測。

適用場景: 當數據模式複雜或無法用線性模型擬合時，決策樹回歸提供了靈活的預測方法。

特性:
優點 缺點
能夠捕捉複雜的數據模式，無需假設數據分佈。 容易過擬合，尤其是當數據較少時。
容易解釋，可以視覺化。 對數據的小變化較為敏感，預測結果不夠平滑。
Request Example:
curl -X POST -H "Content-Type: application/json" -d "{\"data\": [10, 20, 30, 40, 50]}" https://my-api-project-83a1e4f4e00f.herokuapp.com/decision_tree_regression
{
    "max_profit": 17.32,
    "max_revenue": 1385.86
}
4. 隨機森林回歸 (Random Forest Regression)

描述: 隨機森林回歸通過多棵決策樹的平均結果來進行預測，減少過擬合問題。

適用場景: 當數據異質性較高或存在異常值時，隨機森林回歸能提供穩定的預測。

特性:
優點 缺點
比單棵決策樹更穩定，泛化能力更強。 模型較為複雜，難以解釋。
適合異質性數據和大數據集。 計算成本較高，訓練時間較長。
Request Example:
curl -X POST -H "Content-Type: application/json" -d "{\"data\": [10, 20, 30, 40, 50]}" https://my-api-project-83a1e4f4e00f.herokuapp.com/random_forest_regression
{
    "max_profit": 25.0,
    "max_revenue": 2352.65
}
5. 支持向量回歸 (SVR, Support Vector Regression)

描述: 支持向量回歸是一種基於核函數的非線性模型，能夠擬合高維數據。

適用場景: 當數據非線性且模式複雜時，SVR 提供了高精度的預測能力。

特性:
優點 缺點
能夠處理高維和非線性數據。 模型難以解釋，訓練時間較長。
擬合精度高，適合複雜數據。 需要調整多個參數，調參難度較大。
Request Example:
curl -X POST -H "Content-Type: application/json" -d "{\"data\": [10, 20, 30, 40, 50]}" https://my-api-project-83a1e4f4e00f.herokuapp.com/svr_regression
{
    "max_profit": 25.0,
    "max_revenue": 2352.65
}
